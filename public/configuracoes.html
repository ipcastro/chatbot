Personalidade do seu Bot

Aqui você pode definir uma instrução de sistema personalizada para o seu bot. Ela terá prioridade sobre a configuração global.

Como testar

  1. Crie um usuário via endpoint protegido /api/user/register (use token ADMIN_BOOTSTRAP_TOKEN no header).
  2. Preencha Usuário/Senha acima e clique em "Carregar atual" para ver a instrução salva.
  3. Edite e clique em "Salvar Personalidade". Depois envie mensagens para /chat usando as mesmas credenciais (Basic Auth) para validar.
